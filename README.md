# inept

An "**in**teractive **e**ditable o**pt**ions" library for handling complex option hierarchies
from multiple sources (script, CLI, config files, ...).

## Installation 

Install inept with pip

```bash 
  pip install inept
```

## Running Tests

To run tests, run the following command

```bash
  pytest
```
